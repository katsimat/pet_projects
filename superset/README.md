# Аналитический дашборд пользовательской активности и выручки
[Ссылка на дашборд](https://superset-edu.tbank.ru/superset/dashboard/p/DLkXneajoQb/)

Этот проект представляет собой интерактивный дашборд для анализа пользовательского поведения, выручки и эффективности воронки продаж.

---


## Метрики на графиках:

1. **Выручка**
2. **Выручка по неделям**
3. **MAU за последние 30 дней**
4. **WAU за последние 7 дней**
5. **DAU за вчера**
6. **MAU**
7. **WAU**
8. **DAU**
9. **Доля сессий по источникам**
10. **Доля сессий по источникам по неделям**
11. **Доля сессий по ОС**
12. **Доля сессий по ОС по неделям**
13. **Количество покупок**
14. **Количество покупок по неделям**
15. **Средний чек**
16. **Средний чек по неделям**
17. **ARPU**
18. **ARPPU**
19. **ARPU по неделям**
20. **ARPPU по неделям**
21. **Воронка покупок**
22. **Конверсии в каталог / корзину / покупку по неделям**

---

## Настройки и фильтры

- **Фильтр по периоду**: доступен для всех графиков, **кроме**:
  - MAU за последние 30 дней
  - WAU за последние 7 дней
  - DAU за вчера

- **Фильтр по источнику трафика и ОС**: применяется ко **всем графикам**

- **Выбор грануляции (день/неделя/месяц)**: доступен для всех графиков, **кроме**:
  - MAU
  - WAU
  - DAU
