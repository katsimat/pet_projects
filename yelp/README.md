# Анализ ресторанов и отзывов на платформе Yelp

Проект посвящён исследованию открытых данных Yelp с целью выявления городов и районов с наибольшей концентрацией и качеством заведений.

---

## Цель

- Найти город с наибольшим количеством компаний.
- Внутри этого города выявить районы с наиболее качественными услугами.
- Найти рестораны с наивысшими оценками по отзывам.

---


## Основные шаги проекта

- Подсчёт количества компаний в каждом городе → выбор города `N` с максимумом.
- Отбор всех компаний в городе `N`, визуализация на карте (долгота/широта).
- Группировка компаний по округлённым координатам региона.
- Расчёт средней оценки и количества отзывов в каждом районе.
- Отбор только стабильных регионов (с 10+ компаниями).
- Построение цветных географических графиков по средней оценке.
- Поиск ресторанов с наивысшими оценками на основе `stars`.

---


## Итог

- Определён город с наибольшей деловой активностью.
- Выделены районы с устойчиво высокой пользовательской оценкой.
- Составлен список лучших ресторанов по отзывам.
