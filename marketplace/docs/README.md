# Marketplace Platform

## Описание проекта

Учебный веб-маркетплейс. Проект включает базовую функциональность для размещения товаров.

## Бизнес-модель

**Маркетплейс** — платформа для обмена товарами между пользователями:
- **Продавцы** размещают товары и управляют заказами
- **Покупатели** ищут и приобретают товары

## Технический стек

### Backend
- RESTful API с аутентификацией и авторизацией
- Управление пользователями, товарами, заказами

### Frontend
- SPA с адаптивным дизайном
- Интерфейс для покупателей

### DevOps
- Автоматический деплой через CI/CD

## Функциональность

- **Регистрация/авторизация** с ролевой моделью
- **Каталог товаров** с базовым поиском
- **Корзина и оформление заказов**
- **Личные кабинеты** для пользователей
